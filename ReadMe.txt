培训的最后一题，有关ADC/DAC/DMA的使用。

所以最后一题预留了较多的测试时间。
而今晚也是刚把做好的部分测了一遍，
给我测试的是一个学姐，人很好，非常温柔也很可爱。
今晚上想说的则是又一次体会到了请教是多么重要，
在测试ADC/DAC功能时，波形一直有问题，

在信号发生器和示波器这两个玩意上捣鼓半天，又在代码上debug了半天，
实在找不到错误，心态临近崩溃，最后找到硬件的学长请教，

我顿时释然：原来不是代码的问题，突然又庆幸还好请教了学长，
不然自己在这折腾半天，心态可能就崩了。
现在渐渐能感受到实验室那种学习氛围了，
有不懂的不羞于启齿，多与他人交流，才能够学习到更多的东西。

实在撑不下去了，在十一点半时便回去了。
